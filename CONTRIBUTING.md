# Contributing

Thanks for your interest in contributing!

## Development setup

### Requirements

- Node.js (for `pnpm` / `npx`)
- `pnpm`
- Rust toolchain (`rustup`)

### Install

```bash
pnpm install
```

### Run (development)

```bash
pnpm tauri dev
```

## Pull requests

- Keep changes focused and small.
- If you change behavior, update README / docs accordingly.
- Prefer clear commit messages.

## Code style

- TypeScript/React: keep components small and readable.
- Rust: keep error handling explicit; avoid panics in runtime paths.
